# 📍 Java Location App (Android)

This is a simple Android application built using Java that fetches the user's current location and converts it into a human-readable address.

---

## 🚀 Features

- 📌 Get current latitude and longitude
- 🏠 Convert coordinates to address (Reverse Geocoding)
- 📱 Simple and clean UI
- ⚡ Works without Google Maps API

---

## 🛠️ Tech Stack

- Java
- Android SDK
- LocationManager (GPS)
- Geocoder (for address conversion)

---

## 📂 Project Structure

- `MainActivity.java` → Handles location fetching and address conversion
- `activity_main.xml` → UI layout

---
## ⚙️ Permissions Used

<uses-permission android:name="android.permission.INTERNET"/>
